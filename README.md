# Kaustuv Dey — Portfolio

Personal portfolio website built with modern web technologies.

**Live:** kaustuv.vercel.app

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Language:** TypeScript
- **Icons:** Lucide React

## Features

- Dark / Light mode with system preference detection
- Cursor glow effect that follows mouse movement
- Smooth scroll navigation
- Auto-scrolling photography marquee with lightbox
- Expandable research paper abstracts
- Fully responsive across all devices

## Sections

- Hero with social links
- About & Skills
- Education
- Work Experience (timeline)
- Projects (with GitHub links)
- Achievements & Publications
- Photography
- Contact


## Deployment

Deployed on [Vercel](https://vercel.com). Push to `main` triggers automatic deployment.
